# Basic Calculator

##Features
- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (√∑)
- Real-time result display while typing
- AC (Clear All) and C (Delete one character) buttons
- Supports keyboard input:
  - Number keys
  - Operator keys (+, -, *, /)
  - Enter for equals
  - Backspace to delete
  - Escape/Delete to clear
